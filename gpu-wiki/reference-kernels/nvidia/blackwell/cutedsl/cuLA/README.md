# Blackwell CuTeDSL cuLA Kernels

CuTeDSL reference kernel implementations for the cuLA (CUDA Linear Algebra) project on the Blackwell (SM100) architecture.

---

| Kernel | Description |
|--------|-------------|
| [blackwell_fused_fwd.py](blackwell_fused_fwd.py) | Blackwell fused forward kernel |
| [chunk_delta_h.py](chunk_delta_h.py) | Chunk Delta H computation |
| [fwd_o.py](fwd_o.py) | Forward output computation |
| [inv.py](inv.py) | Matrix inversion kernel |
| [kda_fully_fused_wip.py](kda_fully_fused_wip.py) | KDA fully fused kernel (WIP) |
| [la_decode.py](la_decode.py) | Linear Attention Decode |
| [lightning_attn.py](lightning_attn.py) | Lightning Attention implementation |
| [linear_attn.py](linear_attn.py) | Linear Attention implementation |
