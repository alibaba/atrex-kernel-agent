# Blackwell Reference Kernels

A collection of GPU reference kernels for the NVIDIA Blackwell (SM100) architecture, organized by DSL framework.

---

| Directory | Description |
|------|------|
| [cutedsl/](cutedsl/) | CuTeDSL framework kernels (CUTLASS, CuTeX, cuLA, Flash Attention, FlashInfer) |
| [gluon/](gluon/) | Gluon-compiled kernels (TCGen05, CLC, Multi-CTA) |
| [triton/](triton/) | Native Triton kernels (cuLA) |
