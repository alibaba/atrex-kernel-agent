# Hopper Reference Kernels

A collection of GPU reference kernels for the NVIDIA Hopper (SM90) architecture, organized by DSL framework.

---

| Directory | Description |
|-----------|-------------|
| [cutedsl/](cutedsl/) | CuTeDSL framework kernels (CUTLASS, Flash Attention, FlashInfer) |
| [gluon/](gluon/) | Gluon-compiled kernels (TMA, WGMMA, Warp Specialization) |
