# NVIDIA Reference Kernels

A collection of NVIDIA GPU reference kernels organized by architecture generation, covering Ampere, Hopper, Blackwell, Blackwell Ultra, and Blackwell GeForce.

---

| Directory | Description |
|------|------|
| [ampere/](ampere/) | Ampere architecture kernels (CuTeDSL, Gluon, Triton) |
| [hopper/](hopper/) | Hopper (SM90) architecture kernels (CuTeDSL, Gluon) |
| [blackwell/](blackwell/) | Blackwell (SM100) architecture kernels (CuTeDSL, Gluon, Triton) |
| [blackwell-ultra/](blackwell-ultra/) | Blackwell Ultra (SM103/B300/GB300) kernels (CuTeDSL) |
| [blackwell-geforce/](blackwell-geforce/) | Blackwell GeForce/workstation (SM120) kernels (CUDA, CuTeDSL, Triton) |
