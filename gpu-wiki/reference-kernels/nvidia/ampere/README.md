# Ampere Reference Kernels

A collection of GPU reference kernels for the NVIDIA Ampere architecture, organized by DSL framework.

---

| Directory | Description |
|-----------|-------------|
| [cutedsl/](cutedsl/) | CuTeDSL framework kernels (CUTLASS, Flash Attention) |
| [gluon/](gluon/) | Gluon-compiled kernels (Triton) |
| [triton/](triton/) | Native Triton kernels (DeepGEMM) |
