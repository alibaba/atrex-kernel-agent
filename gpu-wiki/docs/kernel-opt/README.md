# GPU Optimization Knowledge

A GPU kernel programming and optimization knowledge base, covering vendor-agnostic general optimization theory, AMD CDNA architecture optimization, and NVIDIA GPU architecture optimization.

---

| Directory | Description |
|------|------|
| [generic/](generic/) | Vendor-agnostic GPU general optimization theory: memory hierarchy, execution model, instruction optimization, application-level optimization |
| [amd/](amd/) | AMD GPU optimization: general framework, CDNA3 (MI300X) and CDNA4 (MI355X) architecture-specific optimization |
| [nvidia/](nvidia/) | NVIDIA GPU optimization: general architecture features, Blackwell knowledge cards, CUTLASS/CuTeDSL/PTX reference, Hopper (sm_90) specific optimization |
